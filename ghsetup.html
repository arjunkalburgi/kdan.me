This is how you can use github to run your own website!

  1. First make a new repository on github.com. Your website will take the url "username.github.io/repository_name"
  2. Create a new branch with the name "gh-pages". At this point you may make it the default branch if you want and delete "master" but you don't have to.
  3. Next, clone the repository to your computer.
  4. Switch the branch in the github app to be gh-pages. This option is near the top left of the app
  5. Navigate to the cloned repo on your computer. In this folder, create a file named "index.html"
  6. Make the website however you want, when you are done head back to the github app. Commit and sync your changes
  7. Your website is now published as you have made it!

To view the url where your website is published at, navigate to the repo on github, then click settings.
About halfway down, it will tell you "Your website is published at xxx"

To change the url where your site is published, you need to buy a domain name. Once you have acquired one, simply make a new file called CNAME and put your domain in there, then commit and sync
